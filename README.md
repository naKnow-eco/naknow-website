# naKnow Website

The official website for [naKnow](https://naknow.eco), a collaborative initiative building an open environmental knowledge base for the electronics industry.

Built with [Nuxt 4](https://nuxt.com).

## Development

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm generate
```
